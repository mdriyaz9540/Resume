# Interactive Resume Dashboard | Power BI

## Overview

This project is an interactive resume dashboard built using Power BI to showcase my professional profile, technical skills, projects, education, and work experience in a visually engaging format. The dashboard transforms a traditional resume into an interactive portfolio that enables recruiters and hiring managers to quickly assess my qualifications and project experience.

---

## Features

* Professional Profile Overview
* Interactive Skills Visualization
* Project Portfolio Section
* Education Timeline
* Professional Experience Summary
* GitHub, LinkedIn, and Resume Navigation Buttons
* Clean and Recruiter-Friendly Dashboard Design
